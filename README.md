# API Testing with Python (Requests)

This project demonstrates API testing using **Python** and the **requests** library in Visual Studio Code (VS Code).  
It includes tests for:
- Get Token
- Verify Base URL
- Create Booking
- Get Booking
- Update Booking
- Delete Booking
